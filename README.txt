It contains 10 datasets, source code and a brief summary of performance evaluation.

The source code of this program is written in python 3. Please run this program directly and it shows the performance of prediction for 10 different datasets.

It contains the functions to calculate two parameters - that is mean and variance - of the predictive distribution on new point. It also has the function to evaluate the performance for 10 datasets in this directory.

The allowed range of input variables is less than 22. This program uses 10 input variables to predict, but you can change the value of variable N in the program to change the range.
